Stock Portfolio Optimization

Combining Modern Portfolio Theory, Black-Litterman Model, and Technical Analysis with AI

Project Overview

This project implements an advanced portfolio optimization strategy that combines traditional financial models with AI-driven analysis. The system analyzes major tech stocks and cryptocurrencies, providing comprehensive insights and optimal portfolio allocation recommendations.

Key Features:

  • Technical Analysis Indicators (RSI, MACD, Bollinger Bands)
  • Modern Portfolio Theory Implementation
  • Black-Litterman Model Integration
  • AI-Powered Market Analysis
  • Dynamic Portfolio Rebalancing

Technical Analysis Components

RSI (Relative Strength Index)

Momentum indicator measuring speed and magnitude of recent price changes to evaluate overbought or oversold conditions.

Bollinger Bands

Volatility indicator using moving averages and standard deviations to identify potential trading opportunities.

MACD

Trend-following momentum indicator showing relationship between two moving averages of asset's price.
