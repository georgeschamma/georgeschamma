Project Overview

This project demonstrates the application of data science techniques to financial analysis and portfolio optimization. By combining statistical methods with artificial intelligence, I've created a comprehensive system for data-driven investment decisions.

Data Analysis Pipeline

Created comprehensive data processing workflows to analyze financial time-series data with Python, including data cleaning and normalization techniques.

Statistical Modeling

Implemented technical indicators and financial models including RSI, MACD, and Bollinger Bands for quantitative market analysis.

AI-Enhanced Insights

Integrated natural language processing to interpret numerical results and generate actionable recommendations from complex financial data.

Technical Analysis Components

RSI (Relative Strength Index)

Implemented a statistical momentum indicator that measures the magnitude of recent price changes to evaluate overbought or oversold conditions.

Bollinger Bands

Developed a volatility indicator using moving averages and standard deviations to identify statistical outliers in price movements.

MACD Analysis

Created a trend-following momentum indicator that reveals changes in the strength, direction, and duration of trends through algorithmic analysis.

Technical Implementation

This project combines five key components using Python libraries:

  • Data Retrieval & Technical Analysis: Using yfinance, pandas, and matplotlib to fetch market data and implement RSI, Bollinger Bands, MACD, and other indicators
  • Portfolio Optimization: Implementing both Modern Portfolio Theory and Black-Litterman models with SciPy and PyPortfolioOpt
  • AI Integration: Using LangChain and OpenAI to generate insights from financial data

The project demonstrates how statistical methods combined with AI can transform complex financial data into actionable investment recommendations.
